Typing Speed

A typing speed checker is a tool that measures how fast and accurately you can type, usually in words per minute (WPM), by having you type a set of words within a time limit.


Hangman game

Hangman is a word-guessing game where players guess letters to uncover a hidden word. Incorrect guesses add parts to a stick figure, and the game ends if the word is guessed or the figure is fully drawn.


Memory Cards

The memory card game is a matching game where players flip over pairs of cards, trying to find matches. The game helps improve memory and focus, and the player with the most matched pairs wins.
